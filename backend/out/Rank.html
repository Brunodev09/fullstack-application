Class: Rank

Rank(users, top) → {list}

Receives a list of user objects from the database and the top5 songs list, which is represented by each song id as a string. Instatiates a new class that represents a user, but associates points to each of them, accordingly to the number of matches in their (users) own songs array and the top5 songs array. As we had no need to mantain any states inside the class, I made the method static, so we can share it through all instances and it becomes a pure function without the need of the "this".
