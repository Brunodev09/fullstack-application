Class: exports

exports(controllers)

The Server class is responsible for initializing the Express framework, the Mongoose and MongoDB drivers, all the middlewares and all of the controllers as well. The MVC/MC pattern allows a pretty decent scalability of controllers with zero code on this class and a one liner on the applications entry file (index.js). Also unifies MongoDB connections, making the use of models really quick by discarding the necessity to establish a new connection at every import of each model. Also responsible for shutting the service down if anything goes wrong.
